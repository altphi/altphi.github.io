
For a long time I've preferred terminal apps/utilities over web based or native
alternatives.  They are typically faster, simpler, keyboard navigable,
configurable, and hackable/OSS friendly.  Here's my current "stack."

- aerc     - email
- kubetui  - kubernetes tui
- lazygit  - git visualizer
- ncspot   - music (spotify)
- qalc     - calculator
- todui    - (my own conconction) todo / focus / time tracking
- peaclock - clocks and timers
- nvim     - editor
- tmux     - my [helper scripts] have made tmux use a lot of fun... I can jump straight to any buried terminal, resize quickly etc etc, practically a window manager within a window manager.
- btop     - system stats
- zathura  - ok this isn't really tui but it's a pdf viewer fully keyboard driven and vim-like in every way
